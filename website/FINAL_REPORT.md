## 🚀 部署指南

### 方式一：Vercel (推荐)
```bash
# 1. 连接GitHub仓库
# 2. 导入项目到Vercel
# 3. 自动检测Next.js配置
# 4. 一键部署

vercel
```

### 方式二：手动部署
```bash
# 构建生产版本
npm run build

# 将 .next 和 public 目录部署到服务器
# 启动生产服务器
npm start
```

### 环境变量
```env
NEXT_PUBLIC_SITE_URL=https://cook-hire.com
NEXT_PUBLIC_SITE_NAME=Cook-Hire
```
