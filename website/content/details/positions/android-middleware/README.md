# Android 车载中间层工程师

> 招聘人数: 2人 | 状态: 🟢 招聘中

## 职位概述

我们正在寻找 **Android 车载中间层开发工程师**，负责设计和开发车载应用的中间层架构，使各类应用能够统一接入并运行在该层之上。

## 核心职责

1. **中间层架构设计** - 设计可扩展的车载应用中间层架构
2. **应用适配框架** - 开发应用安装、运行、管理的统一框架
3. **系统集成** - 与车载系统底层（如 Android Automotive）深度集成
4. **性能优化** - 确保中间层的稳定性和高性能
