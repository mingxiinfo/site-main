# Android高级开发工程师（车载应用）

## 岗位概述

有潜力、学习能力强的Android高级工程师，负责应用层功能开发、车机系统适配和集成工作。

## 基本信息

- **招聘人数**：1人
- **年资要求**：3-5年（快速成长型）
- **薪资范围**：25-30K/月 + 绩效奖金
- **工作地点**：杭州余杭区（主要），联调期短期出差上海
- **项目周期**：6个月

## 核心要求

### 必须精通（P0）
- Java/Kotlin编程基础
- Android四大组件和生命周期
- 多线程编程（Handler、协程）
- 网络编程和数据存储

### 优先考虑（P1）
- 车载或嵌入式Android开发经验
- 了解Android Framework基础
- 插件化框架使用经验
- 强烈的学习意愿和技术深度追求
