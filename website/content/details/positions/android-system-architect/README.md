# Android系统架构师（车载沙盒方向）

## 岗位概述

资深Android系统架构师，负责车载应用沙盒管控系统的架构设计和核心技术攻坚。

## 基本信息

- **招聘人数**：1人
- **年资要求**：10年+（8年+ Android开发，3年+ Framework经验）
- **薪资范围**：30-35K/月 + 绩效奖金
- **工作地点**：杭州余杭区（主要），联调期短期出差上海
- **项目周期**：6个月

## 核心要求

### 必须精通（P0）
- Android Framework（AMS、PMS、WMS等系统服务）
- Binder/AIDL进程间通信和Hook技术
- Native开发（C/C++、JNI/NDK）
- Linux操作系统基础

### 优先考虑（P1）
- VirtualApp等虚拟化框架经验
- AOSP定制或ROM开发经验
- 车载系统（Android Automotive）经验
